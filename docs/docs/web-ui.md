# Web UI

Access: `http://localhost:7391`

## Features

- Create/start/stop/delete workspaces
- Web terminal
- View logs
- Configure credentials
- View AI agent sessions

## Workspaces Page

- Click "+" to create
- Click workspace to view details
- Click "Terminal" to open shell

## Settings

- **Environment** - Add env vars
- **Files** - Add credential files
- **Agents** - Configure AI assistants
